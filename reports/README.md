# Meridian Reports

Git-tracked operational reports generated by Meridian's scheduled agents.
Browsable at `brain.markahope.com/reports/`.

```
reports/
├── lint/          ← weekly wiki health checks (linter agent)
├── evolution/     ← weekly knowledge drift detection (evolution detector)
└── layer4/        ← monthly Layer 4 conceptual reports
```

Reports are written by agents running on the VM. The `vm-auto-deploy.sh`
script checkpoints this directory before each `git reset --hard` so
reports survive code deploys. Syncthing replicates them to the local
vault; periodic `git add reports/ && git commit` preserves them in
version history.
